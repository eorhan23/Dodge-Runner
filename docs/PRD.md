MVP (İlk Sürüm)

1. Karakter Hareketi

* Sabit x-konumunda sürekli koşma
* Yukarı Ok tuşu ile zıplama
* Aşağı Ok tuşu ile eğilme

2. Engeller

* Üstten gelen alçak engel (eğilme gerektirir)
* Zeminden yükselen engel (zıplama gerektirir)
* Çift engel / kombinasyon (zıplama + zamanlama gerektirir)
* Farklı hızda hareket eden engel

3. Zorluk Artışı

* Zamanla artan oyun hızı
* Zamanla artan engel sıklığı

4. Skor Sistemi

* Hayatta kalınan süreye veya geçilen engel sayısına dayalı skor
* Skorun oyun sırasında ekranda gösterilmesi

5. Oyun Sonu

* Çarpışma ile oyunun bitmesi
* Final skorun gösterilmesi
* Tekrar Oyna seçeneği
